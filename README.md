#Android万能遥控器
主要功能介绍：
1. 遥控电脑，ppt播放,无线调试
2. 红外遥控空调等其他家电
3. 植入百度语音唤醒，用户呼唤“ok，小浩” 说出对应指令。
4. 指纹登录，数据备份还原等。
